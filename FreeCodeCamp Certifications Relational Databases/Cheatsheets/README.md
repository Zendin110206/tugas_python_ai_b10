# Relational Databases Cheatsheets

This folder stores quick lookup references for syntax learned in the
freeCodeCamp Relational Databases Certification track.

The detailed workshop notes and `*_Syntax_Reference.md` files remain in their
original topic folders. This folder is different: it is meant to be opened when
I need to quickly remember how to write a command, query, condition, loop, join,
or import pattern.

## Files

| File | Focus | Best used when |
| --- | --- | --- |
| `Bash_Cheatsheet.md` | Bash terminal and scripting syntax. | Writing shell scripts, loops, conditions, input handling, CSV parsing, and `psql` automation. |
| `PostgreSQL_SQL_Cheatsheet.md` | PostgreSQL and SQL syntax. | Creating databases, designing tables, querying rows, filtering with `WHERE`, grouping, joining tables, and exporting dumps. |

## Scope Policy

These cheatsheets are based on material already practiced in this repository.
When new freeCodeCamp Relational Databases topics are completed, new syntax can
be added here without changing the older workshop notes.

## How to Use

1. Open the topic file that matches the problem.
2. Find the section by goal, not by theory.
3. Copy the syntax pattern.
4. Replace table names, column names, variables, or values with the current
   problem context.

Example:

```text
Need to filter rows by text?
Open PostgreSQL_SQL_Cheatsheet.md -> WHERE Conditions -> Text filters.
```
